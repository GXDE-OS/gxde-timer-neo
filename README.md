<h1 align="center">一只番茄</h1>


### 🍭 编译

#### 依赖安装
```
sudo apt install qt5-qmake qttools5-dev-tools libdtkwidget-dev debhelper libdtkwidget5 libdtkgui5 libdtkgui5 fakeroot
```

#### 编译
```
dpkg-buildpackage -b -d
```
